# How to contribute to this Project?

This is the thesis of the CAS Modern Software Engineering & Development of the Lucerne University of Applied Sciences. Due to this we unfortunately cannot accept any contributions from others.

## Project setup

```
.
├── .github/
│   └── workflows/              # GitHub Actions
├── .vscode/
│   └── settings.json           # Shared VS Code settings
├── poc/                        # Proof of concept documentation
│   └── README.md
├── thesis/                     # Source code ot the thesis written in AsciiDoc
│   └── thesis.md
└── ...
```

For more detailed instructions on the specific sections, please refer to the following documentation files:

- [poc/README.md](poc/README.md)
- [thesis/thesis.md](thesis/thesis.md)
